# 🧓 CARE-O – Senior Citizen Health Care App

CARE-O is a mobile application designed to support the health, safety, and well-being of senior citizens. It helps elderly users stay active, connected, and cared for through simple and user-friendly features.

---

## ✨ Features
- Fitness guidance with different levels  
- Medication reminders with alerts  
- Guardian control for caregiver support  
- Devotional section (prayers, songs, quotes)  
- Music player and simple games  
- Meditation and stress relief techniques  
- Video calling for social connection  

---

## 🛠️ Tech Stack
- Flutter (Frontend)  
- Node.js (Backend)  
- Firebase (Database & Authentication)  
- Firebase Cloud Messaging (Notifications)
